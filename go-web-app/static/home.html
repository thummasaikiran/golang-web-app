DevOps Project

A comprehensive DevOps implementation featuring a Golang web application with end-to-end CI/CD pipeline.

This project demonstrates modern DevOps practices implemented from scratch to production deployment.

Project Overview

This project showcases a complete DevOps workflow built and managed by Sai Kiran Thumma. It includes containerization, orchestration, continuous integration, continuous deployment, and infrastructure as code practices.

The application is a production-ready Golang web service deployed on Kubernetes with automated pipelines for testing, building, and deployment.
